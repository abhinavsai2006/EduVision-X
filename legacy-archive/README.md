# Legacy Archive

This folder stores archived root-level files from the previous Node/Express setup.

## Contents

- `package.json`
- `package-lock.json`
- `server.js`

## Purpose

- Preserve legacy implementation history without cluttering the production workspace.
- Keep the repository focused on the primary `next-app/` architecture.

## Policy

- Do not add new production features in this folder.
- All active development should happen in `../next-app/`.
